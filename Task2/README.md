# Customer Segmentation Analysis using K-Means

## Project Overview
This project performs customer segmentation analysis using the K-Means clustering algorithm.  
The goal is to group customers based on their purchasing behavior to help businesses improve marketing strategies, customer satisfaction, and decision-making.

---

## Objectives
- Analyze customer purchasing patterns
- Segment customers into meaningful groups
- Apply K-Means clustering
- Visualize customer segments
- Generate actionable business insights
